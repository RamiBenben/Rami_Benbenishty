##To add a link that downlaods a certain file follow these steps: 
  1. Exit this file and go to the <a href="Files">Files folder </a>
  2. Just underneath the top-right of the screen there will be a button that say: "Add file"
  3. Click that button and then click "Choose file"
  4. Once you have clicked that a window will appear where you can choose the file you want others to be able to downlaod
  5. After you picked your file and the window closed, click the Commit Changes button (You might need to scroll down)
  6. Open the Folder with the name of the page you want the link to be on (SCBV for School climate, bullying and  violance)
  7. Then Paste this in and change the bold text:
     \<a href="../Files/__your file name.your file ending__" downlaod> __This is what the button will say__\</a>
